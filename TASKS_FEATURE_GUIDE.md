## Available Task Types

| Task Type | Description | Example |
|-----------|-------------|---------|
| `tag-exists` | HTML tag must exist | `<h1>`, `<form>`, `<table>` |
| `html-contains` | HTML code contains text | `type="email"`, `required` |
| `css-contains` | CSS code contains property | `padding`, `display: flex`, `@keyframes` |
| `js-contains` | JavaScript contains code | `addEventListener`, `map`, `=>` |
| `function-exists` | Function is defined | `function add()`, `const myFunc =` |
| `class-exists` | CSS class is styled | `.box`, `.container` |
| `no-starter` | Code differs from starter | Any changes to the template |
